Installation

Samane a été conçu dans l'optique d'être facilement installable.

Installation de Samane

Nous vous recommandons d'installer la version de PHP >= 7.1, MySQL et Composer

php >= 7.1 
Mysql 
Composer (for dependencies)

Pour créer votre projet, tapez la commande suivante:

composer create-project samane/samanemvc your_project_name
